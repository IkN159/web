## Новостной портал — это учебный новостной портал, который показывает, как быстро разработать функциональный сайт с авторизацией, добавлением новостей и удобной админ-панелью.
Пользователи могут не только читать, но и оставлять свои заметки, делая ленту живой и разнообразной.
Стек технологий
*   Технология	- Зачем используется
*   Python	- Серверная логика
*   Flask	- Микрофреймворк для веб‑приложения
*   WTForms	- Создание и валидация форм
*   SQLAlchemy	- Работа с базой данных (ORM)

## Как запустить:
1. Склонируй репозиторий:
    git clone https://github.com/твой-логин/название-репозитория.git
    cd название-репозитория
2. Создай и активируй виртуальное окружение:
    py -m venv .venv
    .venv\Scripts\activate
3. Установи зависимости (все они перечислены в requirements.txt):
    pip install -r requirements.txt
4. Настрой переменные окружения:
    Создай файл .env в корневой папке.
5. Запусти приложение:
    flask run
